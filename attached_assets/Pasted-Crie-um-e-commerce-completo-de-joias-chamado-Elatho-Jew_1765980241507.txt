Crie um e-commerce completo de joias chamado "Elatho Jewels" usando React + TypeScript + Vite + Express.

ESTRUTURA:
- Frontend React em /client
- Backend Express em /server
- TailwindCSS para estilização
- lucide-react para ícones
- react-router-dom para rotas

PÁGINAS OBRIGATÓRIAS:

1. HOME (/)
   - Hero com logo Elatho
   - Grid de produtos (3 colunas)
   - Filtro por categoria
   - ProductCard: imagem, nome, preço, botão ver detalhes

2. PRODUTO (/produto/:id)
   - Imagem grande
   - Descrição completa
   - Seletor de variação (Dourado/Prateado/Rosé)
   - Quantidade
   - Botão adicionar ao carrinho

3. CARRINHO (/carrinho)
   - Lista produtos
   - Editar quantidade
   - Remover item
   - Total
   - Botão finalizar compra

4. CHECKOUT (/checkout) - 3 ETAPAS:
   Etapa 1: Nome, WhatsApp, Email
   Etapa 2: CEP (buscar via ViaCEP), Endereço, Frete (PAC R$15 / SEDEX R$25)
   Etapa 3: Pagamento (PIX 5% desconto / Crédito / Débito)
   Ao finalizar: enviar para WhatsApp e redirecionar para /sucesso

5. SUCESSO (/sucesso)
   - Mensagem de confirmação
   - Botão voltar e WhatsApp

6. ADMIN (/painel-elatho-2025)
   - Login senha: "Elatho@2025!Admin"
   - Dashboard: cards de estatísticas
   - Aba Produtos: listagem
   - Aba Pedidos: listagem com filtros

CARTCONTEXT:
- Gerenciar carrinho (add, remove, quantity, total)
- Persistir em localStorage
